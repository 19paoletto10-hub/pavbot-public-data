## Executive Summary

Ta runda przyniosła kilka istotnych zmian. Największa wartość to nowe,
kanoniczne źródła pracodawców dla Sii, Capgemini i CDQ oraz nowa, bardzo
praktyczna rola NUMLABS skoncentrowana na produkcyjnych agentach i RAG.
Dodatkowo People More pokazuje dziś inną, lokalną stawkę niż w baseline, a
bezpośrednia strona KYOTU ujawnia ważny niuans: to obecnie bardziej talent pool
na nadchodzące projekty niż jednoznacznie natychmiastowy headcount projektowy.

## Top New Or Materially Changed Roles

### 1. Sii - AI Engineer – LLM / Agentic Systems (f/m/x)

- Dlaczego interesujące: to jedno z najmocniejszych publicznych ogłoszeń w tej
  rundzie dla praktycznej pracy przy agentach LLM i systemach retrieval.
- Fit LLM/AI: Python, OpenAI, LangChain / LangGraph / LlamaIndex, vector
  databases, RAG, software engineering w środowisku produkcyjnym.
- Lokalizacja/remote: Polska, wiele lokalizacji; remote / hybrid / office.
- Wynagrodzenie: brak widełek w publicznym źródle.
- Niepewność: publiczny listing nie pokazuje pensji; lokalizacja jest
  wielomiejscowa, a nie wyłącznie Wrocław.
- Źródło: [Sii](https://sii.pl/en/job-ads/id/37154/ai-engineer-llm-agentic-systems-f-m-x)

### 2. Sii - Senior Machine Learning / LLM Systems Engineer (f/m/x)

- Dlaczego interesujące: rola jest bliżej AI platform/inference niż aplikacji
  demo; dotyczy wydajności, skalowania i pracy na realnym sprzęcie.
- Fit LLM/AI: distributed training and inference, Python, C++, CUDA, Triton,
  Kubernetes, optymalizacja throughput/latency dla dużych systemów ML/LLM.
- Lokalizacja/remote: Polska, wiele lokalizacji; remote / hybrid / office.
- Wynagrodzenie: brak widełek w publicznym źródle.
- Niepewność: źródło nie rozbija lokalizacji na sam Wrocław; trzeba traktować
  jako remote-Poland / multi-location.
- Źródło: [Sii](https://sii.pl/en/job-ads/id/37017/senior-machine-learning-llm-systems-engineer-fmx/?location=PL_CRACOW)

### 3. Capgemini - Senior Software Engineer (Agentic AI, Cloud-native, On-prem)

- Dlaczego interesujące: rzadkie publiczne ogłoszenie z Wrocławia, które wprost
  łączy agentic AI, RAG, CI/CD, integracje eventowe i produkcyjne wdrożenia od
  PoC do MVP i skali.
- Fit LLM/AI: LLM integrations, RAG, vector search, prompt orchestration,
  Azure OpenAI / OpenAI / open-source models, Docker / Kubernetes.
- Lokalizacja/remote: Wrocław; model hybrydowy.
- Wynagrodzenie: brak widełek w publicznym źródle.
- Niepewność: część zakresu jest architektoniczno-konsultingowa, choć nadal
  wyraźnie hands-on.
- Źródło: [Capgemini](https://careers.capgemini.com/job/Wroc%C5%82aw-Senior-Software-Engineer-%28Agentic-AI%2C-Cloud-native%2C-On-prem%29/1388839033/)
