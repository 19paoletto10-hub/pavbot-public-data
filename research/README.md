# Research Topics

Each topic folder is a durable workspace for one research thread.

## Required Files

- `topic.md` - topic contract and research scope.
- `index.md` - current state of knowledge.
- `backlog.md` - follow-ups and review notes.
- `runs/` - dated daily reports.
- `proposals/` - risk-gated proposed changes.
- `podcasts/` - optional dated podcast packages for topics with audio output.

## Naming

Use lowercase slugs with hyphens, for example `codex-agent-automation`.
The folder name should match the topic thread purpose.
Use ASCII slugs even when the content is Polish, for example `polska-swiat`.
